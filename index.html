Computer Graphics - Amusement Park Project

This project was done for the CS447P/557 Introduction to Computer Graphics class at Portland State Univerisity. The assignment was to create an amusement park from scratch using a framework called openGL. We were provided a flat terrain and a b-spline for a roller coaster. Our goal was to create several objects in the environment that would be seen at an amusement park and generate them with openGL. All aspects including texturing and positioning were part of the project. Users could also add custom models created in Blender, but would have to write a class to interpret the files.
